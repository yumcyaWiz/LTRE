# LTRE

header only montecarlo raytracer for my research

## Requirements

* C++17
* OpenMP

## Build

```
mkdir build
cd build
cmake ..
cmake --build .
```

## Externals

* [assimp](https://github.com/assimp/assimp)
* [spdlog](https://github.com/gabime/spdlog)
* [GoogleTest](https://github.com/google/googletest)
